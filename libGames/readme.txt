No diretório criado, digite os comandos para compilar a biblioteca:

mkdir build
cd build
cmake ..


Importante: assume-se que o diretório está no mesmo nível do diretório "libIIA" 
(que contém os arquivos da biblioteca de IIA). Por exemplo:
/workspace/libGames
/workspace/libIIA
